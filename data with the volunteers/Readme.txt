The file names are set according to “gender_initials”. Twelve healthy 22 to 35-year-old participants (nine males and three females) took part in each experiment. As for the in-hand device, the tests under each condition (3 interaction methods and 4 stiffness conditions) were randomly applied and recorded for 5 times. As for the stepping device, the tests under each condition (4 stiffness conditions) were randomly applied and recorded for 3 times. The mat. files show the mean value of the identification rates for each participant.
